Request Header Parser Microservice

Cách sử dụng:

Gửi GET request đến /api/whoami

API sẽ trả về thông tin về request của bạn bao gồm:

  • IP address của bạn
  • Ngôn ngữ ưa thích của trình duyệt
  • Thông tin về trình duyệt/software

Ví dụ kết quả:

{
    "ipaddress": "123.45.67.89",
    "language": "en-US",
    "software": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) ..."
}
